### ALSUA Automata - Instalación y Uso

1. Instala Python 3.10+ y pip
2. Instala librerías: pip install -r requirements.txt (por definir)
3. Renombra `.env.example` a `.env` con tus credenciales
4. Ejecuta con: python main.py

El script procesa correos, lee archivos .xls, llena el CRM y registra resultados en MySQL o CSV.
